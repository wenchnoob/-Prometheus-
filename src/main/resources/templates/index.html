Welcome to project Prometheus.


This is a website created by Wenchy Dutreuil, primarily for fun and personal development (but this might go to the moon so stay tuned). If you have any questions and/or concerns feel free to reach out to me at my email wcdutreuil@gmail.com.



Usage


This website (at the moment) is intended to be used as an online calculation resources. Some details are listed below. There will be some extensions to this website in the coming months. Navigate pages using the options in the top right. The should all be pretty self-explanatory. Good luck, and have fun. Don't be shy about sending me feedback wcdutreuil@gmail.com.



Tooling/API's

  • Spring Boot
  • MySQL
  • Thymeleaf
  • GraphQL

Features

  • Can utilize the calculator to perform basic computations.
  • You can sign up for an account.
  • You can login to an existing account.
  • You can logout from an account one you have logged in.
  • ANY THING NOT MENTIONED ABOVE IS NOT YET SUPPORTED.
