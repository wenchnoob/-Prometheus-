Intro

This is a general purpose (minimilast) calculator. Expression's are parsed using an abstract syntax tree (AST) and are then evaluated in order using java built ins. Yes, this project is implemented in java. To use this just type in the expression to be evaluated and then hit enter and VOILA!

Supported Operations

  • Addition "+"
  • Subtraction "-"
  • Multiplication "*"
  • Division "/"
  • Exponentiation "**"
